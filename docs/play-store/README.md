# Google Play Store — BuzzOff materials

Ready-to-use copy and assets for the Play Console listing. Use [buzzoff.me](https://buzzoff.me) as reference.

| File | Use in Play Console |
|------|----------------------|
| **store-listing.txt** | Short description (80 chars) + Full description |
| **privacy-policy.html** | Host at `https://buzzoff.me/privacy` and paste that URL in Store listing → Privacy policy |
| **feature-graphic.html** | Open in browser, screenshot or export at 1024×500 for Feature graphic |
| **screenshots-guide.md** | Which app screens to capture for Screenshots |
| **icon-512.md** | How to get the 512×512 app icon |

After adding materials, complete the rest of the [publishing checklist](../PUBLISH_GOOGLE_PLAY.md#checklist-first-release).
