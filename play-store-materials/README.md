# BuzzOff — Google Play Store materials

All assets and copy for the Play Console listing in one folder.

| File | Use |
|------|-----|
| **store-listing.txt** | Short description (80 chars) + Full description → Main store listing |
| **privacy-policy.html** | Host at `https://buzzoff.me/privacy` → Privacy policy URL |
| **feature-graphic.html** | Open in browser, resize to 1024×500, screenshot → Feature graphic |
| **screenshots-guide.md** | Which app screens to capture → Phone screenshots |
| **icon-512.md** | How to get 512×512 PNG → App icon |

Full publishing steps: **docs/PUBLISH_GOOGLE_PLAY.md**
